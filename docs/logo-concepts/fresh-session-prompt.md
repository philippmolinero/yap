# Prompt for a Fresh Claude Code Session

Paste this into a new Claude Code session in the yap project directory:

---

```
Use the /canvas-design skill to create a logo mark and app icon for "Yap" — a macOS menubar voice dictation app. Hold a key, talk, release, and your words appear wherever you're typing. Minimal, utilitarian, stays out of your way.

Brand personality: quiet, confident, minimal. Like Raycast, Things 3, Bear, Linear. No decoration. No corporate feel. Indie Mac app energy.

Color direction: PASTEL. Soft, muted, calming. Think dusty pastels — not saturated, not candy-bright. Explore palettes like:
- Soft lavender / periwinkle on deep muted navy
- Dusty rose / blush on warm dark gray
- Pale mint / sage on slate
- Soft peach / apricot on charcoal
- Muted sky blue on dark indigo

Pick the palette that feels most right for a voice/speech tool and commit to it.

Logo mark direction: something that communicates "voice" or "speech" without being literal or generic. Avoid: generic microphone icons, basic speech bubbles, Wi-Fi/sound wave clichés. Instead explore more abstract, distinctive interpretations — a form that you'd recognize at 16px in a macOS menubar and that feels like it belongs next to icons by indie Mac developers who obsess over every pixel.

The final output should be a macOS app icon (512x512 or 1024x1024) with the macOS superellipse rounded corners. The mark should work in monochrome for the menubar template icon.

Create something that a designer would actually ship — not placeholder art.
```
