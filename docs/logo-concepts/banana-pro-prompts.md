# Banana Pro Prompts for Yap Logo

## Prompt 1 — App Icon (primary)

```
A minimal macOS app icon for a voice dictation app called "Yap". Flat design, no gradients. Warm clay/terracotta color (#C27B5A) mark on a dark charcoal (#2D2926) background with rounded superellipse corners (macOS icon shape). The mark is a stylized speech bubble with an audio waveform carved into it — 5 vertical bars of varying height inside the bubble, like a voice visualizer. The bubble has soft rounded corners and a small organic tail at the bottom-left. Clean, geometric, utilitarian aesthetic. Inspired by the design language of Raycast, Things 3, and Linear app icons. No text, no shadows, no 3D effects, no gloss. Professional app icon design, vector-style, centered composition.
```

## Tips for iterating in Banana Pro

- If the result is too complex, add: "Simpler. Fewer elements. More negative space."
- If it looks too generic, add: "Make it more distinctive and memorable. Less corporate, more indie."
- If colors are off, be explicit: "Use ONLY these two colors: #C27B5A and #2D2926"
- To test readability, ask: "Show this icon at 16px, 32px, 128px, and 512px side by side"
- For variations: "Generate 4 variations of this icon, each with a different abstract interpretation of voice/speech"
